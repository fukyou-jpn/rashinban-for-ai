# プライバシーポリシー（Rashinban for AI）

最終更新日：2026年6月21日

## このツールについて

Rashinban for AI（以下「本拡張機能」）は、Webページ上の操作可能なUI要素（CSSセレクタ・座標・状態など）を抽出し、お使いのコンピュータ上で動作するローカルのMCPサーバーへ受け渡すChrome拡張機能です。AIエージェント（LM Studio等）がブラウザを操作する際の補助を目的としています。

## 収集する情報と、その用途

本拡張機能が扱う情報は、すべて**お使いのコンピュータ内部に閉じています**。外部の第三者サーバーへ送信されることはありません。

| 情報 | 保存場所 | 用途 |
|---|---|---|
| 認証トークン | ブラウザの拡張機能ストレージ（`chrome.storage.local`）、およびローカルファイル（`config.json`） | 拡張機能とローカルMCPサーバー間の通信を認証するため |
| 拡張機能ID | ローカルファイル（`config.json`） | 正規の拡張機能からの接続であることを確認するため |
| 閲覧中ページのURL・タイトル | 送信のみ（保存はしません） | ローカルMCPサーバーが、どのタブに対する操作要求かを判別するため |
| ページ内の操作可能要素（CSSセレクタ・座標・要素の状態） | 送信のみ（保存はしません） | AIエージェントがページを操作するための情報として、ローカルMCPサーバー経由で提供するため |

## 第三者への提供

本拡張機能は、収集した情報を広告・分析・その他の第三者サービスへ提供しません。外部APIとの通信も行いません。

## 通信の範囲

本拡張機能とMCPサーバー間の通信は、`ws://127.0.0.1:3765`（localhost）に限定されています。インターネットを経由した通信は発生しません。

## データの削除

- 認証トークンは、ブラウザの拡張機能の設定からいつでも削除・変更できます。
- ローカルに保存された `config.json` は、ファイルを直接削除することで初期化できます。

## 権限について

本拡張機能は、以下の権限を使用しています。

| 権限 | 用途 |
|---|---|
| `scripting` | ページ内の操作可能要素を抽出するため、コンテンツスクリプトを実行します |
| `tabs` | 現在アクティブなタブの情報を取得するため |
| `clipboardWrite` | 抽出した要素情報をJSON形式でクリップボードにコピーする機能のため |
| `storage` | 認証トークンを保存するため |
| `host_permissions`（`ws://127.0.0.1/*`） | ローカルMCPサーバーとのWebSocket通信のため |
